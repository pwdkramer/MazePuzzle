# MazePuzzle
Given a Board, a starting tile, and a destination tile, find the shortest path and directions.

Example Board:
Puzzle = [
 ['-', '-', '-', '-', '-'],
 ['-', '-', '#', '-', '-'],
 ['-', '-', '-', '-', '-'],
 ['#', '-', '#', '#', '-'],
 ]
 
 You can move through tiles with a '-' but not a '#'.
